# WBC 2026 データストア

**2026 ワールドベースボールクラシック（WBC）** の打席・投球レベルのデータセットです。[MLB Stats API](https://statsapi.mlb.com/api/v1) から収集し、**Tableau でのデータ分析・可視化**を目的として整備しています。

> ラウンド終了後に随時更新しています。
> 最終更新: 2026-03-20

## このデータセットの使い方（Tableau）

1. `csv/` フォルダ内の CSV ファイルをすべて Tableau に読み込む
2. `試合ID` をキーにテーブルを結合する
3. 試合・イニング・打席・投球の各粒度で分析できる

```
試合マスタ（wbc2026_games）
  └─ 試合ID で結合
       ├── イニング別スコア（wbc2026_innings）  → イニング別得点推移・累積スコア
       ├── 打席結果（wbc2026_atbats）           → 打者・投手別成績
       ├── 投球詳細（wbc2026_pitches）          → 球種・球速分布
       └── 走塁イベント（wbc2026_events）       → 盗塁・暴投などのイベント

選手プロフィール（wbc2026_players）
  └─ 投手 / 打者 の選手名で結合
       ├── 打席結果（wbc2026_atbats）
       └── 投球詳細（wbc2026_pitches）
```

---

## Tableau での計算フィールド例

打席結果（wbc2026_atbats）の bool カラム（1/0）を使って、以下の指標が計算フィールドで表現できます。

| 指標 | 計算式 |
|---|---|
| 打率（AVG） | `SUM(安打) / SUM(打数)` |
| 長打率（SLG） | `SUM(塁打数) / SUM(打数)` |
| 出塁率（OBP） | `SUM(安打 + 四球 + 死球) / SUM(打席)` |
| OPS | 長打率 + 出塁率 |
| 空振り率 | `SUM(スイング - コンタクト) / SUM(スイング)` |

## テーブル別の分析テーマ

| 分析テーマ | 主に使うテーブル |
|---|---|
| 投手分析（球種・球速・奪三振） | 投球詳細 + 選手プロフィール |
| 打者分析（打率・長打率・OPS） | 打席結果 + 選手プロフィール |
| 試合展開（得点推移・逆転） | イニング別スコア |
| 国別比較（勝敗・得失点） | 試合マスタ |

## データセット概要

| ファイル | 行数（目安） | 説明 |
|---|---|---|
| `csv/wbc2026_games.csv` | 約 50 | 試合マスタ（スコア・ラウンド・チーム） |
| `csv/wbc2026_innings.csv` | 約 500 | イニング別ラインスコア（累積スコア含む） |
| `csv/wbc2026_atbats.csv` | 約 4,000 | 打席結果（bool フラグ・塁打数含む） |
| `csv/wbc2026_pitches.csv` | 約 16,000 | 投球ごとのデータ（球速・球種・結果グループ含む） |
| `csv/wbc2026_events.csv` | 約 500 | 打席外イベント（盗塁・暴投など） |
| `csv/wbc2026_players.csv` | 約 300 | 選手プロフィール（身長・体重・所属 MLB チームなど） |

全ファイルは `試合ID` または選手名をキーに結合できます。

## CSV スキーマ

### 試合マスタ（wbc2026_games.csv）

| カラム | 型 | 説明 |
|---|---|---|
| 試合ID | 数値 | MLB Stats API の gamePk |
| 日付 | 文字列 | 試合日（YYYY-MM-DD） |
| ラウンド | 文字列 | Pool A〜D / Quarterfinal / Semifinal / Final |
| プール | 文字列 | グループ（A/B/C/D）。ノックアウトは空欄 |
| アウェイチーム | 文字列 | アウェイチーム名 |
| ホームチーム | 文字列 | ホームチーム名 |
| アウェイ得点 | 数値 | アウェイ得点 |
| ホーム得点 | 数値 | ホーム得点 |
| 勝利チーム | 文字列 | 勝利チーム名 |
| 敗戦チーム | 文字列 | 敗戦チーム名 |
| 得点差 | 数値 | 勝利チーム得点 − 敗戦チーム得点 |
| 試合時間（分） | 数値 | 試合時間 |
| ステータス | 文字列 | Final / Completed Early |
